Call:

call() method calls a function with a given this value and arguments provided individually.it invokes a function with a specified context

        var obj={
            name:'hello',
        }
        function Details(a){
            return this.name+a;
        }
        Details.call(obj,100)

Apply:

Apply method accepts an array of arguments apply invokes the function immediatly and allows you to pass in arguments as an array.

   var obj={
        name:'hlo',
    }
    function Details(a){
        return this.name+a;
    }
    Details.apply(obj,[100])

Bind:

Bind returns a new Function and you can invoke/call it anytime you want by invoking a function

        var obj={
            name:'hi',
        }
        function Details(a){
            return this.name+a;
        }
        var fun = Details.bind(obj,[100])
        fun();

Prototype:

Whenever we create an object or array or function, our js engine return an object to our original object.
